Role Name
===========

This roles will intall open5gs EPC core and configure network configuration for attaching radio.

Requirements
-------------
* Ubuntu 20.04
* Ansible 2.10.8
* Vagrant 2.2.19
* MongoDB:		6.0.11
* Mongosh:		2.0.2
